## 2. Canvas 覆盖层与绘制

- [x] 2.1 在原句卡片外层设 `position: relative`，内叠 `canvas`（`absolute inset-0`），`pointer-events` 随 `doodleMode` 切换
- [x] 2.2 实现 Pointer Events：`pointerdown` 开始路径、`pointermove` 连线、`pointerup/cancel` 结束；`touch-action: none` 防止滚动吞手势；每笔完成后压入**笔划栈**（点列或等价），支持整画布重绘
- [x] 2.3 按容器尺寸与 `devicePixelRatio` 设置 `canvas` 分辨率，避免模糊；监听 `resize` / `ResizeObserver` 必要时重绘或清空并提示（见设计）
- [x] 2.4 默认笔划样式：合适线宽与颜色，在浅色/暗色下可读

## 3. 与现有句子交互隔离

- [x] 3.1 `doodleMode === true` 时，原句文本区域对句子逻辑关闭 `pointer-events`（或等价短路），确保点滑不触发划成分/批注/查义
- [x] 3.2 `doodleMode === false` 时恢复现有行为；确认与现有 `dragState`、模态打开状态无死锁

## 4. 生命周期与清理

- [x] 4.1 提供「清除涂鸦」动作，清空笔划栈并 `clearRect`
- [x] 4.2 提供「撤回上一笔」：弹栈并重绘（无笔时 noop 或禁用按钮）
- [x] 4.3 在 `onBeforeUnmount` 与句子弹窗/路由离开时清除涂鸦状态与画布；不调用任何保存涂鸦的 API
